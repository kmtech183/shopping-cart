A fully functional e-commerce shopping cart application built with React, featuring product browsing, cart management, and persistent storage. This project was created as part of [The Odin Project](https://www.theodinproject.com/lessons/node-path-react-new-shopping-cart) curriculum.

## ✨ Features

- 🏠 **Home Page** - Welcome section with featured features and call-to-action
- 🛍️ **Shop Page** - Browse products fetched from FakeStore API
- 🛒 **Shopping Cart** - Add/remove items, update quantities, view totals
- 💾 **Persistent Storage** - Cart data saved in localStorage
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 🎨 **Modern UI** - Beautiful gradients, animations, and hover effects
- 🧪 **Comprehensive Testing** - Unit and integration tests with Vitest
- 🚀 **Performance Optimized** - Built with Vite for fast builds and hot reload

## 🛠 Technologies Used

### Core

- **React 19** - UI library
- **React Router DOM 7** - Navigation and routing
- **Vite** - Build tool and development server

### Styling

- **CSS Modules** - Scoped and maintainable styling
- **CSS3** - Animations, gradients, flexbox, grid

### Testing

- **Vitest** - Unit test framework
- **React Testing Library** - Component testing
- **JSDOM** - DOM simulation

### API

- **FakeStore API** - Product data source

### Development Tools

- **ESLint** - Code linting
- **Prettier** - Code formatting
